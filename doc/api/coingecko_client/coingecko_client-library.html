coingecko_client

coingecko_client library

CoinGecko API client.

Main coingecko_client library

Classes

AssetPlatform
Asset platform information. Use in getting list of an asset platforms or a specific asset platform.
BaseCurrency
Base class for the currency classes.
Currencies and CryptoCurrencies can be used for explicit values while CustomCurrency is used to specify non standard values.
BaseModel
Base class of model classes.
Can be extended to create model class for other data formats.
This provides utiliy methods to explcitly convert data to it's target type to avoid formatting error in runtime
CoinBasicInfo
Contains the coin's basic information.
CoinCategoryBasicInfo
Basic information of a coin category.
CoinCategoryInfo
Coin category detailed info
CoinGeckoClient
Main coingecko_client library class. This is the class that should be instantiated to access all the available coingecko api endpoints.
CoinInfo
Coins detailed information.
CoinLinks
Contains urls from different sources. This is mainly used for the coin result but is also used in other domain.
CoinMarket
Contains the coin market data used when getting all the coin market list or just getting an information about a specific market.
CoinMarketHistory
CoinOhlc
Provides data for the coin's open, high, low and close for a specific period.
CoinTickers
Provides a list of tickers for a specifc coin. Mainly used for retrieving coin data.
CompanyInfo
Crypto company information.
CompanyList
Crypto company listing information.
ContractInfo
A wrapper class of CoinInfo. Used mainly for getting coin info from a contract address.
ContractMarketHistory
A wrapper class of CoinMarketHistory.
Used mainly for getting coin info from contract address.
CustomCurrency
You might need to use currencies that does not exist
in either Currencies or CryptoCurrencies enum/class. You can use this do define custom currencies.
Derivatives
Derivatives main information. Used in obtaining list of derivatives or a specific derivatives.
DerivativesExchange
Derivatives exchange main information.
DerivativesTickerInfo
Derivatives ticker detailed information.
ExchangeBasicInfo
Exchange basic information. Used when returning list of exchanges.
ExchangeInfo
Exchange main information. Provides detailed information of a specific exchange.
ExchangeRate
Exchange rate data. Used when collecting rates of a specific coin.
ExchangeStatus
Exchange status. Mainly used in ExchangeInfo class.
ExchangeVolumeChart
Wrapper class of HistoricalData class. Used specifically for exchange volume.
GlobalCryptoInfo
GlobalDefiInfo
HistoricalData
Used by several classes to represent collection with date and value. It could be price, ticker or some other data that requires this format.
MarketData
MarketIndex
Market index information. Use in getting the list of indexes or a specific market index.
NftBasicInfo
Nfts basic information. Use to obtain basic info of all the NFT ids and other nft related information.
NftInfo
Nft information. Contains all information of a specific nft.
Project
Roi
Basic roi information. Used mainly for coin information.
TickerInfo
Ticker detailed information. Used in coins and other domains.

Enums

AssetPlatformsFilter
Mainly use for filtering asset platform result list.

Use to define sorting order of coin categories result.

CoinPriceChange

CoinTickersDataOrdering
Defines the sorting order of coin tickers result
CryptoCurrencies
Collection of known cryptocurrencies.
Currencies
Collection of known world currencies.
DataRange
Use to specify data for a specific date ranges.
DerivativesExchangeOrdering
Use to define sorting order of derivatives exchange.
DerivativesTickers

ExchangeDataOrdering
Use to define sorting order exchange data.
NftsDataOrdering
Use to define sorting order of nft result.

Exceptions / Errors

DataParsingException
This exception is raised when issue occured during the parsing of the json data.
Data can be unreadable or have an incorrect formatting.
NetworkRequestException
Used to raise when a network issue has occurred while sending the request.
to the server. Request and response will be loogged as a result which help to determine the main cause of the error.
